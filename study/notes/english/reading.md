# 一、考研英语一阅读 核心定位
1. 分值：**4篇阅读，每篇5题，共20题，40分**
考研英语**得阅读者得天下**，阅读决定过线、高分。
2. 文章来源：外刊精读
《经济学人》《卫报》《纽约时报》《大西洋月刊》等，**学术议论文为主**。
3. 题材分类
- 社会文化
- 经济商业
- 科技科普
- 教育心理
- 法律政治
4. 文章特点
**长难句多、逻辑绕、观点隐晦、爱考同义替换、态度立场、段落逻辑**。

# 二、做英语一阅读 必须掌握的 6 大核心能力
## 1. 词汇能力（同义替换是阅读灵魂）
考研阅读**不考直译**，考**同义改写**。
同一意思，原文一个词，选项换另一个表达。
必须掌握：
- 考研核心一词多义
- 熟词僻义
- 形容词/动词同义替换
- 态度词、感情色彩词

## 2. 长难句拆解能力
每篇阅读至少 **5–8个长难句**，出题点基本都在长难句里。
必须会：
- 找主干：主谓宾/主系表
- 剥离修饰：定语从句、状语从句、插入语、非谓语
- 读懂倒装、强调、分隔结构

## 3. 段落逻辑能力
考研阅读不靠逐字翻译，靠**逻辑做题**。
必掌握逻辑关系：
- 转折：but / however / yet / nevertheless（**出题最重**）
- 因果：because / therefore / thus / as a result
- 递进：moreover / furthermore / in addition
- 并列：and / also
- 举例：for example / such as（例子为观点服务）

**黄金规则：转折之后必是重点、必出答案**

## 4. 题型识别能力
英语一阅读固定 **6大题型**，每种有固定套路，不用读懂全文也能做题。

## 5. 作者态度&情感判断
必背态度词，直接秒杀态度题：
- 正面：supportive / optimistic / positive / favorable
- 负面：critical / pessimistic / negative / disapproving
- 中立：neutral / objective
- 永远不选：indifferent（冷漠）、biased（偏见）、confused（困惑）

## 6. 排除干扰选项能力
考研选项套路极强，干扰项有固定特征，学会**反向排除**比找正确答案更快。

# 三、英语一阅读 6 大题型 满分做题技巧
## 1. 细节题（占比最多 60%）
特征：问具体事实、原因、时间、人物、做法
做题方法：
1. 题干关键词定位原文
2. 定位句**前后一句**就是答案
3. 选项和原文**同义替换**为正确
4. 拒绝主观脑补，**原文没说绝对不选**

干扰项特征：
- 偷换概念
- 扩大/缩小范围
- 无中生有
- 因果倒置

## 2. 主旨大意题
特征：best title / main idea / mainly about
做题技巧：
1. 看**首尾段** + 各段首句
2. 答案必须覆盖**全文主题**，不能只讲一段
3. 太小、太偏、太绝对直接排除

## 3. 推理判断题
标志：infer / imply / suggest / learn from
核心原则：
**不能选原文直白原话**，要选**合理推导**
语气委婉优先：may / might / likely / partly

绝对化选项必错：must / never / always / only

## 4. 词义句意猜测题
做法：
1. 看单词所在句**前后逻辑**
2. 利用转折、因果、并列关系猜意思
3. 熟词一定考**僻义**，不要选常见意思

## 5. 例证题
标志：example / illustrate / case
核心：**例子本身不重要，例子支撑的观点才重要**
答案一般在：**例子前一句 / 段落首句**

## 6. 态度题
做法：
1. 找作者形容词、副词、转折词
2. 直接匹配态度词
3. 冷漠、偏见、迷惑类永远排除

# 四、标准做题步骤（考场固定流程，直接照练）
## 四步法（一篇阅读固定 15–18 分钟）
1. **先看题干，不看选项**
画出关键词：人名、大写、时间、专有名词

2. **通读全文，抓段落主旨**
不用逐句翻译，抓：首尾段、每段首句、转折词

3. **题干定位，点对点做题**
一题定位一处，不跨段乱找

4. **对比选项，同义替换+排除干扰**
优先排除绝对化、无中生有、偷换概念

# 五、英语一阅读 专门训练方案（怎么练才提分）
## 1. 训练资料
只需要：**考研英语一真题 2005–2025**
不用模拟题，模拟题逻辑和真题差太远。

## 2. 训练节奏（推荐）
### 基础阶段（现在—6月）
- 每天 **1篇 英语一阅读**
- 不追求速度，追求：**逐句精翻、拆解长难句、记生词**
- 每篇做完：精读全文，把所有长难句划主干、翻译通顺

### 强化阶段（7—10月）
- 每天 **2篇**，限时 **18分钟/篇**
- 只按题型套路做题，训练定位+逻辑
- 总结每道题：错因、题型、干扰项套路

### 冲刺阶段（11—12月）
- 整套限时模拟，4篇一起做
- 训练时间分配、心态、快速定位

## 3. 精做一篇阅读的标准流程（提分核心）
1. 定时做题，模拟考场
2. 对答案，不直接看解析
3. 自己复盘：为什么错？定位错？单词不认识？逻辑没看懂？
4. 逐句精读：划从句、非谓语、主干
5. 整理：生词、熟词僻义、同义替换、经典长难句
6. 复盘题型：属于哪类题，下次用什么套路

## 4. 绝对避坑习惯
- 不要逐字翻译全文再做题（浪费时间、容易被细节带偏）
- 不要靠语感脑补，一切答案**忠于原文**
- 不要只刷题不复盘，不复盘等于白做

# 六、阅读必须熟记的做题铁律
1. 转折后必有重点，转折后必有答案
2. 正确选项多为**委婉、概括、同义替换**
3. 错误选项多为**绝对化、无中生有、偷换概念**
4. 主旨看首尾+各段首句
5. 例证题答案在例子前面观点
6. 推理题不选原文原句，选合理推断
7. 态度题优先中立、客观，排除冷漠偏见
