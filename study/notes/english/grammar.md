> 核心说明：考研英语一不考偏难怪语法，只考**长难句拆解能力**，所有语法最终目的：**看懂长难句、会写作文句式**。

---

## 一、基础句法框架（考研一切语法的根基）
### 1. 五大基本句型
所有英语句子都逃不出这5种，长难句都是**简单句叠加修饰**而来。
1. 主谓：S + Vi
   - Birds fly.
2. 主谓宾：S + Vt + O
   - I love learning.
3. 主谓双宾：S + V + IO + DO
   - He gave me a book.
4. 主谓宾补：S + V + O + C
   - We keep the room clean.
5. 主系表：S + LinkV + P
   - She is happy.

**考研关键**：再长的句子，先找出**主谓宾/主系表**，剩下全是修饰成分。

### 2. 句子分类
- 简单句：只有一套主谓
- 并列句：简单句 + 并列连词 + 简单句
  连词：and / but / or / so / yet
- 复合句：主句 + 从句（考研长难句主力）

---

## 二、三大从句（考研重中之重，翻译阅读必考）
### （一）名词性从句
功能：相当于一个**名词**，可作主语、宾语、表语、同位语。
包含：**主语从句、宾语从句、表语从句、同位语从句**

#### 1. 引导词
- that：无含义，不充当成分
- whether/if：是否
- 特殊疑问词：who/what/which/when/where/why/how

#### 2. 必考句型
1. 宾语从句
   - I think **that he is right**.
2. 主语从句
   - **That he succeeded** surprised us.
   - It is obvious that...（形式主语，考研高频）
3. 同位语从句（考研翻译超级爱考）
   跟在抽象名词后：fact / news / idea / belief / suggestion
   - The fact **that time is limited** is obvious.
   区别：同位语从句 that**不做成分**；定语从句 that**做主语/宾语**

### （二）定语从句（长难句第一杀手）
功能：**修饰名词**，翻译成“……的”

#### 1. 关系代词
who / whom / whose / which / that / as
#### 2. 关系副词
when / where / why

#### 3. 考研必考规则
1. that 用法
   可指人/物，不能引导**非限制性定语从句**（逗号后面不用that）
2. 非限制性定语从句（逗号隔开）
   补充说明，去掉不影响主句意思
   - He passed the exam, **which** surprised everyone.
3. as 引导定语从句
   固定：as we all know / as is mentioned above
4. 介词 + 关系代词（考研翻译高频）
   - the person **to whom** you spoke

### （三）状语从句
功能：修饰动词、形容词、整句话，表**时间、原因、条件、让步、目的、结果、比较**

#### 九大常考状语从句及连词
1. 时间：when / while / as / since / until
2. 原因：because / since / as / now that
3. 条件：if / unless / once
4. 让步：although / though / even if / while（尽管）
5. 目的：so that / in order that
6. 结果：so...that / such...that
7. 比较：than / as...as
8. 方式：as / as if
9. 地点：where

**考研重点**：**让步、原因、条件、时间** 四类最多，阅读大量出现。

## 三、非谓语动词（考研写作+长难句核心）
三大类：**不定式to do、现在分词doing、过去分词done**
本质：**不作谓语**，做定语、状语、宾语、补语

### 1. 不定式 to do
- 表目的：To improve English, we need to practice.
- 作后置定语：a way **to learn English**

### 2. 现在分词 doing
- 主动、进行
- 作定语：the girl **singing in the room**
- 作状语：**Hearing the news**, he cried.

### 3. 过去分词 done
- 被动、完成
- 作定语：a book **written by Lu Xun**
- 作状语：**Faced with difficulties**, we should persist.

### 4. 考研必考：独立主格
非谓语自带逻辑主语，和主句主语不一致
- The weather **being fine**, we went out.

### 5. 动名词 doing
相当于名词，可作主语宾语
- **Reading** is beneficial.

## 四、特殊句式（考研翻译、写作必用）
### 1. 倒装句
#### 部分倒装（考研高频）
否定词放句首，助动词/情态动词提前：
Never / Seldom / Hardly / Only / Little
- **Only in this way** can we succeed.（写作万能句）

#### 完全倒装
地点/方位副词放句首：
Here comes the bus.

### 2. 强调句
句型：**It is/was + 被强调部分 + that/who + 剩余部分**
- It is **perseverance** that leads to success.
判断方法：去掉 It is/was...that，句子完整就是强调句。

### 3. 虚拟语气（考研阅读翻译常考，写作可加分）
#### （1）if 条件虚拟
- 与现在相反：If + did, would do
- 与过去相反：If + had done, would have done

#### （2）名词性从句虚拟
suggest / advise / propose / demand / insist 等
从句用 **(should) + 动词原形**
- I suggest that he (should) study harder.

#### （3）固定句型
It is high time that + 过去式  该做某事了

### 4. 省略句
状语从句中**主语一致+含be动词**，可省略主语和be
- While (he was) walking, he thought a lot.
考研长难句经常省略，看懂必须会还原。

## 五、主谓一致 & 代词 & 情态动词
### 1. 主谓一致
就近原则：either...or / neither...nor / not only...but also
就远原则：as well as / together with / rather than

### 2. 代词
不定代词：each / every / all / none / both / either
考研常考：**指代还原**，阅读里找代词指代谁是解题关键。

### 3. 情态动词
must / can / could / may / might / should / ought to
重点：
- could have done 本可以做却没做
- should have done 本应该做却没做

## 六、时态与语态（考研不用死磕，掌握核心即可）
### 1. 必考6大时态
一般现在、一般过去、一般将来、现在完成、过去完成、现在进行
**重点：现在完成时**
have done：过去动作对现在有影响

### 2. 被动语态
be + done
考研特点：学术文章**大量用被动**，翻译常需要转主动中文。

---

## 七、考研英语一语法核心做题逻辑（必记）
1. 拿到长难句 **先找主句主谓**
2. 划出从句引导词：that/which/who/when/because/if
3. 划出非谓语：to do / doing / done
4. 剩下都是插入语、介词短语、修饰成分
5. 先拆主干，再补修饰，翻译就通顺

---

## 八、考研英语一语法终极考点总结（背诵清单）
1. 定语从句（介词+关系代词、非限制性）
2. 名词性从句（同位语从句、形式主语it）
3. 状语从句（让步、原因、条件）
4. 非谓语动词（doing/done/to do 作定语状语）
5. 倒装、强调、虚拟语气
6. 长难句拆分逻辑
