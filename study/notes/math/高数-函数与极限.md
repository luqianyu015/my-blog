# 1. 函数与极限

## 一、核心考点
1.  函数的性质：有界性、单调性、奇偶性、周期性
2.  复合函数、反函数、分段函数
3.  数列极限、函数极限（含左右极限）
4.  无穷小的比较、等价无穷小替换
5.  两个重要极限
6.  极限存在准则（夹逼、单调有界）
7.  函数的连续性与间断点分类
8.  闭区间上连续函数的性质（最值、介值、零点）

## 二、必背公式
1.  两个重要极限：
    \[
    \lim_{x\to 0}\frac{\sin x}{x}=1,\quad
    \lim_{x\to \infty}\left(1+\frac{1}{x}\right)^x=e
    \]
2.  常用等价无穷小（\(x\to0\)）：
    \[
    \begin{aligned}
    &\sin x \sim x,\ \tan x \sim x,\ \arcsin x \sim x,\ \arctan x \sim x, \\
    &1-\cos x \sim \frac{1}{2}x^2,\ \ln(1+x) \sim x,\ e^x-1 \sim x, \\
    &a^x-1 \sim x\ln a,\ (1+x)^\alpha-1 \sim \alpha x
    \end{aligned}
    \]
3.  极限运算法则：若 \(\lim f(x)=A,\lim g(x)=B\)，则
    \(\lim(f\pm g)=A\pm B,\ \lim(fg)=AB,\ \lim\frac{f}{g}=\frac{A}{B}(B\neq0)\)

## 三、解题技巧
1.  求极限“三板斧”：等价无穷小替换 → 洛必达法则 → 泰勒展开
2.  看到 \(1^\infty\) 型，直接用 \(u^v=e^{v(u-1)}\) 变形
3.  数列极限优先考虑：夹逼准则、单调有界准则
4.  间断点分类：先看左右极限是否存在，再看是否相等/等于函数值
