第一部分 入门乐理

目标：看懂六线谱、认识音名、掌握节奏、明白吉他指板基本规律，能弹简单旋律与和弦

1. 吉他基础认知

1.1 标准调弦（6弦→1弦，粗到细）

最常用EADGBE，记忆口诀：饿啊滴鸡逼饿

  • 6弦（最粗）：低音E
  • 5弦：A
  • 4弦：D
  • 3弦：G
  • 2弦：B
  • 1弦（最细）：高音E

木吉他核心特殊点：只有2弦与3弦之间是半音，其余相邻弦都是纯四度，所有指板推算都基于此。

1.2 半音与全音（吉他最小音单位）

  • 半音：相邻1品（吉他最小音高差）
  • 全音：2个半音，相隔1品
  • 自然音固定关系：C-D、D-E、F-G、G-A、A-B 全音；E-F、B-C 半音

1.3 音名与唱名

  • 音名（固定不变）：C D E F G A B
  • 唱名（首调）：1 2 3 4 5 6 7（do re mi fa sol la si）

2. 节奏与节拍（弹唱不抢拍、不拖拍核心）

2.1 常用拍号

  • 4/4拍：以四分音符为1拍，每小节4拍，民谣弹唱90%通用
  • 3/4拍：华尔兹节奏，每小节3拍
  • 2/4拍：进行曲节奏

2.2 基础音符时值

音符类型 时值 4/4拍小节内数量
全音符 4拍 1个
二分音符 2拍 2个
四分音符 1拍 4个
八分音符 1/2拍 8个
十六分音符 1/4拍 16个

2.3 木吉他常用扫弦节奏

  1. 4/4拍万能分解：53231323
  2. 万能扫弦：下 下上 上下上
  3. 抒情慢歌：下 空 下上 上下上

入门必练：开节拍器60BPM，先练稳四分音符、八分音符，节奏稳比弹得快重要100倍。

3. 六线谱识读（吉他专属谱）

  • 6条线，从上到下对应吉他6弦→1弦
  • 线上数字：代表品位，0=空弦
  • 和弦图：上方字母为和弦名，竖线=琴弦，横线=品位，黑点=手指按位
  • 箭头：扫弦方向，向上箭头=下扫，向下箭头=上扫

第二部分 基础乐理

目标：掌握音程、三和弦、调式、和弦进行，脱离死记硬背，能弹唱绝大多数流行歌

1. 音程（和弦的底层逻辑）

1.1 定义

两个音之间的音高距离，单位为度

1.2 吉他必用核心音程

  • 大三度：明亮，大调和弦核心
  • 小三度：暗淡，小调和弦核心
  • 纯五度：和弦稳定音，所有三和弦必备
  • 八度：同音高不同音区，吉他1弦和6弦空弦为八度

2. 三和弦（木吉他弹唱基石）

2.1 和弦定义

3个及以上音，按三度关系叠加，即为和弦

2.2 4种基础三和弦

  1. 大三和弦：根音+大三度+纯五度，标记C，音色明亮稳定，大调主和弦
  2. 小三和弦：根音+小三度+纯五度，标记Cm，音色柔和忧伤，小调主和弦
  3. 增三和弦：根音+大三度+增五度，标记Caug，极不稳定，极少用
  4. 减三和弦：根音+小三度+减五度，标记Cdim，紧张感强，多用于过渡

2.3 C大调开放和弦（木吉他必背）

  • 大三和弦：C G F
  • 小三和弦：Am Em Dm
  • 减三和弦：Bdim

这7个和弦，能弹80%的C调流行民谣弹唱。

3. 调式与级数（乐理核心，告别死记和弦）

3.1 自然大调音阶

全全半全全全半，固定公式，所有大调通用

3.2 和弦级数（万能编配核心）

以C大调为例，级数固定，换调不换级数

  • Ⅰ级：C（主和弦，歌曲开头结尾最常用）
  • Ⅱ级：Dm
  • Ⅲ级：Em
  • Ⅳ级：F（下属和弦）
  • Ⅴ级：G（属和弦，最强倾向主和弦）
  • Ⅵ级：Am（关系小调主和弦，最常用抒情和弦）
  • Ⅶ级：Bdim

3.3 万能和弦进行（直接套用）

  1. 流行万能：Ⅰ-Ⅵ-Ⅳ-Ⅴ（C-Am-F-G）
  2. 民谣经典：Ⅰ-Ⅴ-Ⅵ-Ⅲ-Ⅳ-Ⅰ-Ⅳ-Ⅴ
  3. 伤感抒情：Ⅵ-Ⅳ-Ⅰ-Ⅴ（Am-F-C-G）

4. 小调乐理

4.1 关系小调

大调向下小三度，即为关系小调，共用调号、共用和弦

  • C大调 ↔ Am小调
  • G大调 ↔ Em小调
  • F大调 ↔ Dm小调

4.2 小调和弦进行

万能进行：Ⅵ-Ⅲ-Ⅶ-Ⅳ（对应C调：Am-Em-Bdim-F）

第三部分 进阶级乐理

目标：掌握七和弦、转调、封闭和弦、编配、简易扒歌，达到独立玩琴水平

1. 七和弦（丰富音色，告别单调）

在三和弦基础上叠加七度音，木吉他弹唱进阶必备

  1. 大七和弦：Cmaj7，明亮柔和，多用于抒情歌
  2. 属七和弦：C7，紧张感强，强烈倾向主和弦，万能收尾
  3. 小七和弦：Cm7，忧郁慵懒，R&B、民谣常用
  4. 半减七和弦：Cm7b5，过渡和弦
  5. 减七和弦：Cdim7，极强张力，连环过渡

2. 封闭和弦（横按，全调通用）

2.1 两大核心封闭和弦

  1. E型封闭和弦：以F和弦为基础，整体平移，对应所有大调
  2. Am型封闭和弦：以Fm为基础，整体平移，对应所有小调

2.2 核心逻辑

封闭和弦=和弦形状+根音位置，形状不变，平移即换调，不用再背不同调的开放和弦。

3. 转调（不用变调夹弹全调歌曲）

3.1 变调夹原理

变调夹夹几品，就是升高几个半音，夹2品=D调，夹4品=E调

3.2 近关系转调

最常用，无违和感

  • 属调转调：C→G
  • 下属调转调：C→F
  • 关系小调转调：C→Am

3.3 万能转调和弦

属七和弦（V7），任何调加入V7，可平滑转调到目标调。

4. 和弦编配（自己给旋律配和弦）

4.1 核心规则

  1. 小节内强拍音，匹配和弦内音
  2. 开头结尾用主和弦（Ⅰ级）
  3. 半终止用属和弦（Ⅴ级）
  4. 抒情用Ⅵ级小调和弦，明亮用Ⅰ级、Ⅳ级、Ⅴ级

4.2 简易编配步骤

  1. 确定歌曲调式
  2. 标出每小节强拍音符
  3. 匹配对应级数和弦
  4. 用七和弦、过渡和弦优化音色

5. 扒歌基础（不用谱子自己弹）

  1. 先听歌曲拍号、速度
  2. 确定歌曲调式（找结尾音，多为主音）
  3. 听低音走向，确定和弦级数
  4. 匹配和弦，优化节奏型

第四部分 木吉他乐理高阶内容

1. 顺阶和弦与离调和弦

  • 顺阶和弦：调内7个和弦，稳定和谐
  • 离调和弦：调外和弦，增加色彩，常用二级小大七、降七级和弦

2. 指板音阶系统

  • 五声音阶（民谣、solo必备）
  • 自然大调音阶指型
  • 小调五声音阶（吉他solo万能）

3. 节奏高阶

  • 切音、闷音技巧
  • 附点音符、三连音
  • 复合拍、摇摆节奏

4. 和弦走向创作

  1. 用Ⅰ-Ⅵ-Ⅳ-Ⅴ创作主歌
  2. 用Ⅳ-Ⅴ-Ⅵ创作副歌
  3. 用过渡和弦做桥段，完成完整歌曲编配

吉他乐理学习路线规划

  1. 入门期：背熟弦名、半全音、六线谱、节奏、C调开放和弦
  2. 基础期：掌握级数、和弦进行、小调、分解/扫弦节奏
  3. 进阶期：攻克横按、七和弦、转调、简单编配扒歌
  4. 高阶期：精通音阶、离调和弦、独立创作编曲
